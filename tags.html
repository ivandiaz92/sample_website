Important HTML tags

This page is designed as a quick reference for some of the most common tags covered in Learn enough HTML to be dangerous. In the process of making it, we will learn how to make HTML tables via table and related tags.

The tables below don´t include HTML tags, but they do list some of the most important things.

BLOCK ELEMENTS

TAG NAME PURPOSE
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header format header elements
div division Sets page divisions apart
ol ordered list creates a numbered list
ul unordered list creates a bulleted list
li list creates an element of the list

INLINE ELEMENTS

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code Show text with code font <code>ipconfig</code> ipconfig
span span Explanation to span <span>My name is Ishamel</span> My name is Ishamel

DOCUMENT TAGS

TAG NAME PURPOSE
html html Defines an html document
head head Defines the head of the document
body body Defines de the are of the document´s body
title title> Defines the title of the page
meta meta Defines de set of characters to use
